# 报价单图片保存功能使用说明

## 问题说明
当使用 `file://` 协议直接打开 HTML 文件时，浏览器会因为安全策略阻止图片的导出，出现以下错误：
```
Failed to execute 'toDataURL' on 'HTMLCanvasElement': Tainted canvases may not be exported.
```

## 解决方案

### 方案一：使用"保存为PDF"功能（推荐）
1. 选择产品型号
2. 点击"保存为PDF"按钮
3. 在打印预览窗口中，选择"保存为PDF"选项
4. 选择保存位置和文件名
5. 点击保存

优点：
- 不需要启动HTTP服务器
- 直接使用浏览器内置功能
- 效果稳定可靠

### 方案二：使用HTTP服务器（适合需要图片格式的情况）
1. 运行 `python simple_server.py` 启动HTTP服务器
2. 在浏览器中访问：`http://localhost:8000/报价单.html`
3. 选择产品型号
4. 点击"保存图片"按钮
5. 图片会自动下载到本地

优点：
- 可以直接保存为PNG图片格式
- 图片质量更高

## 功能对比

| 功能 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 保存为PDF | 无需服务器，操作简单，效果稳定 | 需要额外转换为图片 | 大多数日常使用 |
| 保存图片 | 直接生成PNG图片，质量高 | 需要启动HTTP服务器 | 需要图片格式的场景 |

## 常见问题

### Q: 为什么直接打开HTML文件无法保存图片？
A: 浏览器的安全策略限制了本地文件的跨域访问，使用HTTP服务器可以解决这个问题。

### Q: 保存的PDF如何转换为图片？
A: 可以使用以下方法：
1. 使用在线PDF转图片工具
2. 使用Adobe Acrobat等专业软件
3. 使用截图工具截取PDF内容

### Q: HTTP服务器启动失败怎么办？
A: 检查端口8000是否被占用，可以修改 `simple_server.py` 中的 `PORT` 变量，使用其他端口。

## 技术说明

本报价单使用以下技术：
- HTML5 + CSS3 + JavaScript
- Tailwind CSS 用于样式
- Font Awesome 用于图标
- html2canvas 用于截图（需要HTTP服务器）
- 浏览器打印功能 用于PDF导出
